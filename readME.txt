1. Chapter 2: MATLAB code/
  ...angMom33BP: a function that computes the angular momentum associated to each iteration of an integrator;
  ...energyCalc: a function that computes the energy associated to each iteration of an integrator;
  ...exact: a function that computes the exact solution of the integrable problems;
  ...f_rhs: a function that computes the function f on the right-hand side of the 3 body problem;
  ...forwardEuler: the forward Euler integrator;
  ...integrator: a function that allows the user to choose an integrator to perform;
  ...mainCH2: a notebook that runs the simulations in the thesis;
  ...orders: a script that computes the orders of the numerical integrators applied to the harmonic oscillator;
  ...readData: a function that computes a data struct containing all relevant data values for a given problem;
  ...stormerVerlet: the Stormer-Verlet (VTV) integrator;
  ...symplecticEuler: the symplectic Euler (VT) integrator;
  ...yoshida4th: Yoshida's 4th order integrator.


2. Chapter 4: MATLAB code/
  ...Fbox: an alternate distribution function, not used in the thesis;
  ...forwardEuler: see above;
  ...mainCH4: a notebook that runs the simulations in the thesis;
  ...solverMOC: a function that computes the characteristic solution of the Liouville equation;
  ...stormerVerlet: see above;
  ...symplecticEuler: see above,
  ...yoshida4th: see above.
